# Customer Intelligence Dashboard

## Overview

This project analyzes customer behavior using RFM segmentation to identify high-value customers and churn risk segments. The dashboard provides executive insights into revenue contribution, demographics, and product category performance.

## Tools Used

* SQL (MySQL)
* Power BI
* Data modeling and RFM segmentation

## Key Insights

* Champion customers drive the majority of revenue
* Revenue is concentrated in specific product categories and regions
* At-risk customers present retention opportunities

## Project Structure

* `data/` – Raw datasets
* `sql/` – SQL analysis queries
* `dashboard/` – Power BI dashboard file
* `screenshots/` – Dashboard preview
